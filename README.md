FindMyVehicle KTM

Installation and Running Instructions:
No installation is required because this is a front-end web application built with HTML, CSS, and JavaScript.

How to Run from GitHub?
1. Open the GitHub repository link.
2. Click the green *Code* button.
3. Click *Download ZIP*.
4. Extract/unzip the downloaded folder.
5. Open the extracted project folder.
6. Double-click 'index.html'.
7. The application will open in a web browser.
   
Required Files:
Please make sure these files are in the same folder:
'index.html'
'style.css'
'script.js'
'stops.js'
'routes.js'

Requirement:
An internet connection is required because the application uses map, location search, and OSRM routing services.
